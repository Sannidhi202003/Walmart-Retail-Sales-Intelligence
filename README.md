## Overview
An end-to-end retail analytics project analysing 9,969 Walmart transactions across 
3 branches (Yangon, Naypyidaw, Mandalay). Identifies top-selling product lines, peak 
revenue hours, payment method trends, customer ratings, and branch profitability using 
Python and SQL. Results are presented in an interactive Power BI-style dashboard.

## Key Questions Answered
- Which product line generates the most revenue?
- What time of day sees the highest sales?
- Which branch performs best overall?
- What payment method do customers prefer?
- How do ratings vary across product lines?
